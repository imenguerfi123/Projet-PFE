### objective ###
1- Create an ioctl call system for device. 
2- Create a user application that read from module and write into it via the ioctl.

### cmd ###
1- To display the driver variables:
	./test_app.c OR ./test_app.c -g
2- To set the driver variables:
	 ./test_app.c -s
3- To clear the driver variables:
	./test_app.c -c
